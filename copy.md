## Section: The Problem

**Headline:** You wrote a great agent. Now what?

### Column 1 — Open source it

Free. Forever. You ship to GitHub, get stars, watch other people use it without paying you. The hosting, billing, and UI is on the user. Most never run it.

### Column 2 — Build a full SaaS

Six months of work. You learn auth, Stripe webhooks, sandboxing, rate limits, dashboards. Two-thirds of that work has nothing to do with your actual agent.

### Column 3 — Use Pawn

Import your harness. Get a live, paid product in 60 seconds. Spend your time making the agent better — not rebuilding the same infrastructure as everyone else.

---

## Section: How It Works

**Headline:** From harness to paid product in 60 seconds.

### Step 1 — Import

Paste a GitHub URL. Drop in a Claude skill. Upload a Cursor rule. We parse your harness — tools, prompts, memory schema, integrations — automatically.

### Step 2 — Configure

Pick a name, set your price, choose a model. We give you a hosted URL, secure sandbox, OAuth, and Stripe checkout out of the box.

### Step 3 — Get paid

Share your URL. Users pay with credits that work across any agent on Pawn. You get the cut every time someone runs your agent.

## Section: For Builders (Seller Side)

**Headline:** Built for builders, not no-coders.

**Subheadline:** If you can write a Claude skill, you can ship a SaaS on Pawn.

### Feature Grid (6 cards)

**One-click hosting**
Your agent lives at a dedicated URL with SSL, auth, and analytics. Custom domains included.

**Stripe-powered billing**
Credits, subscriptions, or one-time purchases — your call. Payouts via Stripe Connect.

**Sandboxed runtime**
Every session runs isolated. Tools, browser-use, file access — all secured by default.

**Native integrations**
Pawn agents work in Slack, Discord, WhatsApp, and SMS without extra code.

**Memory + history**
Per-user memory and conversation history out of the box. No database setup needed.

**Built-in analytics**
See where users drop off, which tools fire, which prompts win. Optimize without instrumentation.
